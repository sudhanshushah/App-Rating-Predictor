The aim of this project is to predict the rating of a new app that is yet to be launched.

Dataset taken from kaggle.com and consist 267052 data's.



For detailed explanation of the approach and discussion of the results, refer to the .ipynb file.



Contents:



--> Google-Playstore-Full.zip - Data file with information about App Name, Category, Rating, Reviews, Installs, Size, Price, Content Rating, Last Updated, Minimum Version Required, Latest Version and 4 extra Unnamed Columns. 



--> Software_Requirements.txt - Software requirements to execute the code.
